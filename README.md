# سطحة هيدروليك - موقع خدمات النقل الاحترافية

> موقع ثابت احترافي لخدمات نقل السيارات باستخدام السطحة الهيدروليكية مع نظام إدارة محتوى ذكي

## 🌟 المميزات

### التقنيات المستخدمة
- **HTML5 & CSS3**: هيكل وتصميم احترافي
- **JavaScript**: وظائف تفاعلية وحماية المحتوى
- **PWA**: تطبيق ويب تقدمي مع العمل دون إنترنت
- **Netlify CMS**: نظام إدارة محتوى سهل ومرن
- **JSON**: إدارة البيانات بشكل ديناميكي

### المميزات الرئيسية
- ✅ **تصميم متجاوب**: يعمل على جميع الأجهزة (جوال، تابلت، كمبيوتر)
- 🌙 **الوضع الليلي**: تبديل تلقائي وديناميكي حسب الوقت
- 🔒 **حماية المحتوى**: حماية شاملة من النسخ والسحب
- 📱 **PWA**: إمكانية التثبيت والعمل دون إنترنت
- 🚀 **سرعة عالية**: تحسين الأداء والتحميل
- 🎯 **SEO محسن**: تحسين محركات البحث التلقائي
- 📞 **أزرار عائمة**: اتصال مباشر وواتساب
- 🎨 **ألوان احترافية**: تصميم أنيق يناسب القطاع

## 📂 هيكل المشروع

```
/
├── index.html              # الصفحة الرئيسية
├── manifest.json           # PWA Manifest
├── sw.js                   # Service Worker
├── robots.txt             # إرشادات محركات البحث
├── sitemap.xml            # خريطة الموقع
├── _redirects             # إعادة توجيه Netlify
├── _headers               # رؤوس الأمان
├── netlify.toml           # إعدادات Netlify
├── css/
│   └── main.css           # الأنماط الرئيسية
├── js/
│   ├── main.js            # الوظائف الرئيسية
│   ├── protection.js      # حماية المحتوى
│   └── pwa.js             # وظائف PWA
├── data/
│   ├── site.json          # إعدادات الموقع
│   ├── services.json      # بيانات الخدمات
│   ├── buttons.json       # إعدادات الأزرار
│   └── footer.json        # بيانات التذييل
├── admin/
│   ├── config.yml         # إعدادات Netlify CMS
│   └── index.html         # واجهة الإدارة
└── assets/
    └── images/            # الصور والملفات
```

## 🚀 التشغيل والنشر

### 1. تشغيل محلي
```bash
# استخدام خادم بسيط
python -m http.server 8000
# أو
npx serve .
```

### 2. النشر على Netlify
1. رفع الملفات إلى GitHub
2. ربط Repository بـ Netlify
3. النشر التلقائي عند كل تحديث

### 3. تفعيل Netlify CMS
1. تفعيل Netlify Identity في إعدادات الموقع
2. إضافة Git Gateway
3. دعوة المستخدمين للوصول لـ `/admin`

## 🎨 التخصيص

### الألوان
المتغيرات المحددة في `:root` في `css/main.css`:
```css
--primary-color: #1e40af;      /* أزرق احترافي */
--secondary-color: #3b82f6;    /* أزرق فاتح */
--accent-color: #06b6d4;       /* سماوي */
```

### الخطوط
- **الخط الأساسي**: Cairo (Google Fonts)
- **دعم كامل للعربية** مع RTL

### البيانات
جميع المحتويات قابلة للتعديل عبر:
- ملفات JSON في مجلد `/data`
- واجهة Netlify CMS في `/admin`
